FLUXO DE AUTENTICAÇÃO
─────────────────────────────────────────────────────────────────────────────

1. Usuário acessa / (home)
   │
   └─> Sem token? Mostra links públicos
       │
       └─> Clica "Entrar" → /login

2. /login
   │
   ├─> Email + Password
   ├─> POST /auth/login
   │
   └─> Sucesso?
       ├─ localStorage.setItem('techrent_token', token)
       ├─ localStorage.setItem('techrent_user', user)
       └─ router.push('/') → Dashboard/Sidebar aparece

3. MainLayout detecta token
   │
   ├─ Se tem token:
   │  ├─ Mostra Sidebar com navegação por role
   │  └─ Fetch dados com header: Authorization: Bearer {token}
   │
   └─ Se sem token:
      └─ Mostra apenas conteúdo público
