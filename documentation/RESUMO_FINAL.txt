🔐 SEGURANÇA & AUTH
─────────────────────────────────────────────────────────────────────────────

JWT Implementation:
  ├─ Token armazenado em localStorage
  ├─ Auto-incluído em requests (header Authorization)
  ├─ Logout remove token + redireciona
  └─ MainLayout verifica token para Sidebar

Fluxo:
  1. Login → POST /auth/login
  2. Token salvo → localStorage.setItem()
  3. Sidebar aparece → MainLayout detecta token
  4. Navegação por role → Sidebar.js renderiza itens corretos
  5. Logout → localStorage.removeItem() + router.push('/')
